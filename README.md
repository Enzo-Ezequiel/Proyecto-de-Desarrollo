# Este es mi primer repositorio configurado con Docker en WSL2 y Git.
Es un proyecto para extraer texto de un pdf proporsionado por el usuario. Despues se hace un resumen gracias a un modelo de IA.

## Tecnologías usadas:
 - Python
 - UV
 - geminis
 - copilot
 - chatgpt
 - linux (oracle linux 9.5)
 - Docker (Engine nativo en WSL2)
 - Git para Windows
 - Visual Studio Code
 - Base de datos no relacionada MongoDB

## Metodologias:
 - TDD
 - Proyecto dirigido en GitHub
 - CC los 6 primeros principios de 12 factor APP

## Principios de programacion:
- KISS
- DRY
- YAGNI
- SOLID
